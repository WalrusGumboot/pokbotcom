# pokbotcom: een service om een pokertoernooi mee te spelen

`pokbotcom` is een online webservice waarmee gebruikers kunnen verbinden en via een API een programma een pokertoernooi te laten spelen.
Met behulp van sessie-authenticatie kun je een eigen programma op bepaalde endpoints laten reageren om de server je actie te laten weten.
Ondertussen wordt jouw client op de hoogte gehouden van de acties van andere spelers (calls, raises, folds, etc.); hierop kun je handig inspelen.
